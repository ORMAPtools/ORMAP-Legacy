The first two items select the straight or leader arrow styles.

See examples below for items 3-7

Switch arrowheads moves the arrowhead to the opposite end of the arrow.

Cancel stops the drawing process.

Save as default zigzag arrow saves the current width, curve and curve position values as the default in the registry.


Slide toward point or end slides the zigzag portion along the arrow.
Narrower or wider sets the parallel distance between the straight portions of the arrow.
Less curve or more curve sets the amount of curvature in the zigzag portion.
Flip arrows reverses the direction of the zigzag portion.
