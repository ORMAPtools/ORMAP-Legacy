Places a single arrow in one of three styles.

Click where you want the arrowhead to be. Both points will honor snapping if it is turned on. If the arrowhead is not at the cursor you will need to right-click and select Switch Arrowheads. If you save the mxd file it will remember which end the arrowhead belongs on.

Context Menu options. Note the shortcut keys that may be used instead of the menu for some items. The menu will vary depending on which arrow style is selected. Click an image to see the context menu.
