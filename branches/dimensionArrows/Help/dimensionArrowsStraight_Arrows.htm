Place two opposing straight arrows.

Click where you want the arrowheads to be. If the arrowhead is not at the cursor you will need to right-click and select Switch Arrowheads. If you save the mxd file it will remember which end the arrowheads belong on.

Context Menu options. Note the shortcut keys that may be used instead of the menu.

Shorter will shorten the arrows.
Longer will lengthen the arrows.
Flip will make the arrows outside or inside the selected points.
If you click on a line feature the angle of the two arrows will be set perpendicular to the line. Unlock will free the angle, Lock will set it again.
Cancel will stop the process and let you start again.

If the first point is on a map boundary it may choose the wrong scale. Select the scale manually here.
