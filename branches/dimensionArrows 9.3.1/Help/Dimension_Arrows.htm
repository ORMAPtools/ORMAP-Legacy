These tools create Cartographic Line arrows. Land hooks are always placed as subtype 101, Land Hook. Curved arrows are always placed as subtype 134, Bearing & Distance Arrows. Straight Opposing arrows and Single arrows are placed in the current CartographicLine subtype if one is selected or in the appropriate subtype if an Annotation subtype is the current target. After placing the arrow the editing target is set back to what it was before.
 
Click on a tool image for more help on that arrow.
 
 
To install:
  1. Be sure ArcMap is closed.
  2. Double-click the setup.exe file.
  3. Click on next, etc. until it is done.
  4. In ArcMap go to View/Toolbars and check the Dimension Arrow Tools toolbar.
