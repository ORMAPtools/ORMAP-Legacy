Place two opposing straight arrows.

Click where you want the arrowheads to be. The first two clicks will honor snapping, if it is on. The third point determines the height above the line. Snapping is turned off for this point. If you place the annotation first you may click on the center of the text to place the arrows in an exact line.

If the arrowhead is not at the cursor you will need to right-click and select Switch Arrowheads. If you save the mxd file it will remember which end the arrowheads belong on.

You may choose zero to four dashes on the arrows by clicking the numbers 0 through 4. Because Arc does not show the dashes while the arrows are being manipulated the cursor will show you how many dashes will be placed.  

 The overall length of the arrows is shown correctly.

Context Menu options. Note the shortcut keys that may be used instead of the menu.

Shorter will shorten the arrows.
Longer will lengthen the arrows.
Cancel will stop the process and let you start again.

If the first point is on a map boundary it may choose the wrong scale. Select the scale manually here.
